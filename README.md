<h1 align="center">🌶️ Bell Pepper Disease Classification</h1>

<p align="center">
A Deep Learning-based project to detect and classify diseases in bell pepper leaves using CNN.
</p>

<hr>

<h2>📌 Overview</h2>
<p>
Plant diseases affect agricultural productivity. This project provides an automated system 
to classify bell pepper leaf diseases using image processing and deep learning techniques.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>📷 Image-based disease detection</li>
  <li>🤖 CNN-based deep learning model</li>
  <li>⚡ Fast and efficient predictions</li>
  <li>🌱 Useful for smart agriculture</li>
  <li>📊 Model training and evaluation included</li>
</ul>

<hr>

<h2>🧠 Technologies Used</h2>

  <li>NumPy</li>
  <li>OpenCV</li>

<h2>🧪 Dataset</h2>
<ul>

    <ul>
      <li>Healthy</li>

<h2>⚙️ Installation</h2>

<pre>

cd Bell_Pepper_Disease_Classification
pip install -r requirements.txt
</pre>

<h2>▶️ Usage</h2>

<h3>🔹 Train the Model</h3>
<pre>
